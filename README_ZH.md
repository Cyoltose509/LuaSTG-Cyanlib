# LuaSTG Cyanlib

**LuaSTG Cyanlib** 是一个面向 **LuaSTG Sub** 的模块化脚本库，旨在为 LuaSTG 提供更现代化、工程化、可维护的开发体验。  
本库采用 **类 Java / C# 的 Lua 编程风格**，通过模块分层、对象化封装和统一的更新流程，大幅提升 LuaSTG 脚本层的开发规范和可扩展性。

Cyanlib 目前包含三个主模块：

- Core/ —— 游戏脚本核心（较完善）
- STG/ —— 东方 STG 扩展模块（开发中）
- Test/ —— 示例与演示（可直接运行）

运行附带的 exe 即可查看 `Test` 模块的演示内容。

---

# 模块概览

## **Core – 游戏核心模块**

Core 是 Cyanlib 的基础构成模块（完善度较高），包含：

### 渲染由 Camera 主导
摆脱传统"线性渲染流程"，改为 **Camera 捕捉画面 → 渲染输出** 的架构。  
> 当前仍在测试阶段，后续将持续完善。

### 资源对象化封装  
配合 LuaSTG 的底层资源对象化进程：  
- Shader 已完全对象化  
- 其他资源将随 LuaSTG 更新同步完善

### UI 框架（开发中）  
基本架构已建立，会在未来按需求扩充。

### I18n 国际化模块  
Test 中提供完整使用示例。

### MainLoop：统一的游戏更新循环  
将游戏逻辑拆分为可插拔事件，更清晰、更易管理。

### Math 数学库  
包含常用几何、数值、矩阵等工具。

### Object 对象封装  
为 LuaSTG 对象提供更实用、更工程化的扩展功能。

### Render（2D/3D）  
- 2D 渲染稳定  
- 3D 为实验性功能（未来可能增强或调整）

### RNG 随机库  
提供一致、可控的随机数生成接口。

（不包含素材，仅示例加载结构）

# 编程风格规范（类 Java / C#）

Cyanlib 采用（较为）统一风格规范：

| 类型 | 规则 | 示例 |
|------|--------|--------|
| 类名 | PascalCase | `Camera`, `Vector3`, `MainLoop` |
| 静态方法 | PascalCase | `Camera.New()` |
| 实例方法 | camelCase | `camera:setPosition()` |
| 常量 | UPPER_CASE | `EPSILON` |
| 私有变量 | `lowercase` | `cacheValue` |
